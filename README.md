# AWS::Template

configure AWS services for typical web apps with template. the gem provides:

* VPC settings
* Load Balancers
* DNS
* Security Groups
* Key Pairs
* and some helpers

## Installation

Add this line to your application's Gemfile:

    gem 'aws-template'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install aws-template
