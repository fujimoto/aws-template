# AWS::Configure

configure AWS services for typical web apps. the gem provides:

* VPC settings
* Load Balancers
* DNS
* Security Groups
* Key Pairs
* and some helpers

## Installation

Add this line to your application's Gemfile:

    gem 'aws-configure'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install aws-configure
